0614：Netflix奈飞设置简体中文字幕及界面语言的三种方法

2020-06-14
技术相关

➼ 前言

最近几天在 Netflix 奈飞上重温《纸房子》这部我最爱的剧。

由于我用的是台湾节点，导致Netflix的网站界面默认是繁体中文（在语言那里选择中文后），而字幕也只能在繁体中文、英语、西班牙语之间选择，看了几集感觉实在不能再忍。随一番折腾，终于可以看到熟悉的简体中文界面语言和字幕了。

这里做个记录，给有需要的人提供点灵感。

注：如需查阅更多有关奈飞资源及介绍的内容，请访问 👉 ：2020年奈飞完全观看指南

➼ Netflix奈飞中文之谜

默认情况下，Netflix奈飞是可以选择“中文”这个语种的。然而，这里的“中文”其实是有简体中文和繁体中文的区分的，但无法手动选择。因为Netflix会根据你登陆网站时的IP地址自动（强制）给你选择对应的语种。

比如，当你用美国、香港、台湾等IP地址登陆，在个人界面设置语言为“中文”时，默认就是繁体中文。

只有当你用新加坡等IP登陆时选择中文，才能设置和显示为“简体中文”（其实不止新加坡，根据之前测试，其实印度IP也会显示为简体中文，马来西亚IP的话，也是可以的）。

既然Netflix奈飞的中文语言政策是根据IP所在地来决定，那么我们第一个奈飞简体中文解决方案也就呼之欲出了。

➼ 方案一：使用新加坡IP登陆奈飞并设置为简体中文

理论上，这是最简单的解决方案。

只需要你有一个没有被奈飞屏蔽的新加坡原生IP，登陆你的Netflix账号，在“编辑个人资料”页面，打开语言这一栏，选择“中文”，保存即可。

整个过程没有任何技术含量，执行起来行云流水，没有任何反人类的操作步骤，如果你有条件（新加坡IP），这应该是你的首选。

总结一下：

新加坡IP方案优势：

  • 最简化操作
  • 全平台通用（电脑、手机、平板）

新加坡IP方案劣势：

  • 要求新加坡原生IP节点

但如果你不具备如此“完美”的条件，或者不想为了新加坡节点更换你的服务商，别急，我们还有其他解决方案。

➼ 方案二：使用浏览器插件设置奈飞简体中文字幕（及第二字幕）

第二个方案对你的节点IP没有任何要求，但需要使用Chrome（或Chrome内核）的浏览器，所以Mac上能默认播放1080P高清的Safari就无能为力了（这也是我放弃方案二的主要原因）。

如果你并不要求整个Netflix奈飞的网站、手机界面显示简体中文，而只需要字幕有简体中文的话，而且是在电脑端的Chrome浏览器里追剧，那么方案二适合你。

你只需要安装一个叫做 “NflxMultiSubs (Netflix Multi. Subtitles)”的Chrome插件即可。

装好插件，再次打开Netflix上的视频，并把鼠标放在字幕选择的标志上时，就会发现可以选择双语字幕了，并且可以调整字幕位置、大小。

字幕效果可以看下面这张图。如果是对追剧学英语感兴趣的同学，那么这个插件可谓一举两得。

奈飞字幕插件方案优势：

  • 同时显示双语字幕

奈飞字幕插件方案劣势：

  • 限制为电脑端的Chrome浏览器
  • Mac上无法观看1080P清晰度
  • 手机、平板无法通用
  • 语言界面其他内容默认还是繁体中文（剧名等）

如果你想在奈飞全平台显示简体中文，并且不想被限制在Chrome上，那么别担心，我这里还有个好办法可以帮你一劳永逸解决烦恼。

➼ 方案三：POST请求法设置奈飞简体中文字幕及界面语言

最好的总是要压轴出场，我们的Netflix简体中文方案三也是同样道理。

摸索这个方法，是因为去年在销售奈飞账号时（参见帖子0328：Netflix、Spotify会员项目，月入5万+ – 知乎），想提前帮用户设置好简体中文语言。当时用的是Google Cloud Platform + Python Selenium 自动注册的方式，而多方条件所限，只能在美国IP环境下运行程序。当时为了解决简体中文的问题，还特地写了一个“奈飞简体中文中文语言功能备忘录”，后来经过反复尝试中与发现一个完美的解决方案。

这个方案，不但可以在任何IP环境下实现奈飞简体中文设置的效果，还能整合进我的Python程序，不需要任何人工操作。我甚至还把这个功能单独抽出来做成一个自动设置Netflix简体中文语言的Python程序。

不过我们今天不是要讲如何用Python程序来自动注册奈飞账号，所以跳过程序部分，只来看普通用户如何实现这个功能。

放心，这个操作不需要你会Python编程，只需要有一个Chrome浏览器，并按照接下来介绍的步骤一步步来，就肯定能操作成功。

我们假定你奈飞账户目前的语言为繁体中文。

步骤一：打开Chrome浏览器，并登陆你的Netflix账号，点击右上角你的头像，然后点击“管理使用者”（这个部分根据你语言不通，文字有所不同）。

步骤二：在当前页面空白处单击右键，审查元素。在新打开的窗口中顶部选择“网络（Network）”的选项卡，然后点击“XHR”，如下图所示。

步骤三：单击你的头像，在“语言”栏中选择“English”，然后点击“存储”。

这个时候你会在右侧（或者下侧），看到几条网址的请求记录。找到一条pathEvaluator开头的记录，单击，在右侧Headers中往下拉，在最下面的Form Data中，大概第三行，会看到一条记录：param : “en-TW”。

步骤三：复制并修改POST请求数据

在前一步确认无误的情况下，在这条网址上点击右键，选择“复制”，然后选择 “copy as cURL”，具体看下图。

然后打开一个文本编辑器，在空白文档中粘贴。然后搜索"en-TW”,并修改为 “zh-SG”，注意大小写和短横杠都不要错。

修改完毕后，全选，复制刚才修改过的所有内容。

步骤四：发送修改过的POST请求

到这里是最后一步了，你需要的是重新发送一个cURL请求。因为我用的是Mac，而Mac是自带cURL命令的，但是Windows没有，需要自行安装cURL命令工具。安装步骤具体不再阐述，根据网上教程，大概不到10分钟就能配置好。如果你不想麻烦，而且放心的话，还有一些在线发送cURL请求的网站，比如我顺手找到的一个：Post Curl Requests Online （本人不对网站做任何推荐及背书，仅供参考）。

假定你用Mac，或搞定了Win下的cURL命令环境，或者用在线cURL工具。那么接下来，只需要在命令行窗口粘贴在上一步复制到的cURL命令内容，并且按下回车键即可。

在你按下回车键后大概几秒钟，就可以看到窗口吐出了一大堆英文字符一样的东西，理论上，这就代表你已经操作成功了。

这个时候，回到Netflix网站首页，再次刷新页面，你就可以看到整个网站界面语言已经切换为“简体中文“了。

注意：当你重新点击“个人资料”，查看语言时，发现默认是空白的。切记这个时候千万不要再去手动修改为中文，否则就又切换为繁体，而你又得重走一遍刚才的整个流程了。

POST请求方案优势：

  • 一次修改，全平台通用
  • 界面语言、剧名一样为简体中文
  • 对IP没有要求
  • 不需要安装任何插件

POST请求方案劣势：

  • 要求一点点电脑操作技巧

➼ 总结

以上就是设置奈飞Netflix简体中文字幕（或界面语言）的三种方案。

如果你有新加坡、马来西亚IP，那么方案一明显是最容易的。但如果条件不足，而且你平时都用电脑上的Chrome追剧，那么插件法也相对方便，还能顺带学个英语啥的。

如果你对播放平台有要求，比如要在手机、平板、电视盒子上看奈飞，那么就拿出一点点动手操作的实践精神，使用方案三的POST请求大法，可以帮你一劳永逸解决奈飞这个简体中文的心头大患。

最后，祝大家追剧愉快。
