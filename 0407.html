➼ 前言

一个小众而持久的小而美项目。

➼ Dropbox 扩容项目底层商业逻辑

Dropbox扩容项目：通过用户的推荐链接来注册Dropbox新账户并安装Dropbox客户端的方式，帮用户增加Dropbox网盘空间容量的零成本、技术型项目。

➼ Dropbox 扩容项目案例与收益分析

图中宝贝为多SKU产品，标价从¥39到¥89不等。以销量最大的¥39来计算，月销售额超¥9.6万。这个产品，基本零成本。可以理解为月净利润超¥9万。

不过，我有点怀疑这个数据有水分，我们暂且以50%来统计销售额，那么这个链接依然有¥5万/月的净收益。

➼ Dropbox 扩容项目背景简介与用户需求分析

Dropbox是一款国外的文件存储、同步工具，和国内的百度网盘类似，不过在多平台支持、增量同步、多人协作、不限速下载等方面都完胜百度网盘。

Dropbox在国外受众较广，国内用户则普遍集中于职场高端用户，因为这个软件事实上在国内是无法正常访问的。

新注册的Dropbox用户，默认都有2G空间。相对于国内动辄上T的空间，稍显寒酸。如果想要更大空间，一般是需要付费升级的。不过，Dropbox官方还有个长期活动，邀请新用户送500MB空间，而且这个额外赠送的空间容量还是永久有效的。

通过邀请新用户的方式，最高可以升级到18G空间（2G+16G）。对于一般只存在重要文件的用户而言，这个空间基本是够用了。

➼ Dropbox 扩容项目如何操作，是否有进入门槛？

和《SOUL金币项目》的操作思路类似，都是先提取用户的邀请链接，然后通过这个链接进行后续一系列邀请、激活新用户的操作。

对于一个初始空间2G的用户，扩容到18G，需要邀请16*2 = 32个新用户。

这个邀请Dropbox新用户的过程分为两个环节：

  1. 通过用户邀请链接注册新的Dropbox账号
  2. 在电脑上安装Dropbox 客户端，登陆步骤一新注册的账户，成功激活赠送500MB的活动

好消息是，这两个环节，都可以自动化实现。（事实上，自动化程度的高低也是我个人判断项目是否值得进入的最关键参考因素之一）

注册新Dropbox账号环节

这个环节，通过 Python + Selenium 基本可以实现，唯一需要注意的就是，Dropbox官网注册有一个Google的人机验证环节，需要技术上解决下。

如果无法搞定，可以参考 Spotify 项目的解决方案：在手机APP端搞定注册环节，可能就不再有人机验证这个讨厌的环节。甚至进一步，直接提取注册环节的API接口，网络发送请求，一键搞定。

电脑端安装并激活Dropbox软件环节

有个好消息是，这个环节同样可以自动化实现。

去年我曾亲自尝试过解决这个技术难题，后面发现一个解决方案，这里直接分享给有心想进入这个行业的同学们。

技术方案： Linux + Docker + Dropbox

通过 Dropbox 的 Docker 版，自动安装 Linux 版的 Dropbox，安装完毕以后，会给出一个激活链接。直接打开这个链接，然后登陆前面新注册的账号，就成功搞定激活送500MB的环节。

激活以后，直接删掉刚才的 Docker Image，然后重新操作一遍刚才整个流程，即可在同一台Linux服务器上，反复安装、激活Dropbox的500MB赠送空间了。而且，以上整个过程都可以在命令行模式下操作完成，并不需要桌面环境。也就是说，任意一台有管理员权限的普通VPS即可完成。

不懂技术，准备手工人肉操作？好吧，这位同学，我佩服你的勇气和决心。

➼ Dropbox 扩容项目竞争激烈吗，店铺如何运营？

竞争不算激烈。根据我的观察，那个月销量过千的店铺，恐怕存在了很多年了。

除此外，还有一个号称 Dropbox中文站 的山寨版网站，也提供 Dropbox 扩容操作服务。同样存在了很多年，也没见过第二个类似的网站。

综上，这应该是一个非常小众但又相当暴利的项目。

➼ Dropbox 扩容项目有何高级扩展思路？

试试 淘宝店铺 + 公众号 + 网站同步运营吧，多方引流，淘宝或网站成交。

➼ Dropbox 扩容项目是否有其他风险？

好消息是，Dropbox 这个活动长期存在，也很少发生赠送的空间被找回这种事。

但还有个坏消息：淘宝上架这个“Dropbox扩容服务”的宝贝链接，存在较大难度和风险。一不小心，就会面临删除巴链接、扣分、封店的惨烈境遇。拜托，别问我是怎么知道的。

祝你好运。
