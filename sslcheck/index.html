SSLCheck

实时守护您的域名安全。SSLCheck 是一款功能强大的 Python 命令行工具，旨在帮助开发者和运维人员快速检查 SSL 证书状态，预防因证书过期导致的业务中断。

核心功能

  • 🔐 多域名支持: 支持在单次命令中同时检查多个网站的证书状态。
  • 🌍 自定义端口: 完美支持非标准端口（如 example.com:8443）的证书检查。
  • 📅 详尽信息: 清晰展示证书的生效时间、过期时间、剩余有效天数及颁发机构。
  • 🤖 自动化友好: 具备完善的错误处理和日志记录机制，轻松集成至 CI/CD 流。
  • 🌐 智能语言: 支持中英文界面，根据系统语言环境自动切换，使用无障碍。

为什么选择 SSLCheck？

在 HTTPS 普及的今天，SSL 证书管理已成为运维的核心环节。SSLCheck 不仅简单轻便，更专注于解决“痛点”：它能优雅地处理无效域名、连接超时和验证失败等各种异常情况。无需复杂配置，只需一行命令，即可让您的证书风险尽在掌握。
